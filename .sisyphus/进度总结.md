## 整体架构

A 股量化因子研究框架，基于 **Polars + Tushare + pixi**，区分低中高频因子。当前已完成 LFT（低频，日/周/月频）全链路。

```
common/     ← 共享基础设施：storage、calendar、universe、loader、logger
├── lft/    ← LFT 日/周/月频因子 → IC → 回测 → 换手率
├── mft/    ← MFT 分钟级因子（未实施）
└── hft/    ← HFT tick级因子（预留，Tushare不支持）
```

## 技术栈

| 组件 | 选择 | 用途 |
|------|------|------|
| 环境管理 | pixi 0.67 | Python + PyPI 依赖 |
| DataFrame | Polars 1.x | 惰性求值 + 多线程 + Parquet 原生 |
| 数据源 | Tushare 1.4 | A 股日线/分钟线/财务 |
| 存储 | Parquet (PyArrow) | Hive 分区，谓词下推 |
| 回归 | statsmodels | 截面 OLS（中性化 + Fama-MacBeth） |
| 统计 | scipy | Spearman 秩相关 |
