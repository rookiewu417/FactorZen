## 四、后续路线图建议

### Phase 5: 多因子模型（LFT 层面）

> **定位**：从单因子分析跃迁到多因子建模，实现因子组合与组合优化。

| 模块 | 内容 | 优先级 |
|------|------|--------|
| Fama-MacBeth 回归 | 截面 + 时序双阶段，Newey-West 标准误 | 🔴 高 |
| 因子合成 | 等权 / IC 加权 / 主成分 PCA | 🔴 高 |
| Barra 风险模型 | 因子协方差 + 特异性风险 + Ledoit-Wolf shrinkage | 🟡 中 |
| 组合优化 | Mean-Variance + 行业/市值/换手约束 | 🟡 中 |
| 风险平价 | HRP（层次风险平价）| 🟢 低 |

> **前置条件**：需要足够的差异化因子（建议 20+），当前 10 个因子中周频与日频高度相关。

### Phase 6: MFT 分钟频因子完善

> **定位**：将 MFT 从骨架推进到可用状态。

| 模块 | 内容 | 优先级 |
|------|------|--------|
| MFT 因子库 | 8 类 15+ 因子：日内动量 / 波动率 / 偏度 / 价量相关 / 订单流 | 🔴 高 |
| MFT 评估管线 | 高频 IC / 日内回测 / 交易成本模型 | 🔴 高 |
| 微结构处理 | 买卖价差反弹 / 异步校正 / 最优执行 | 🟡 中 |
| L2 数据接入 | 盘口快照 / 逐笔成交 / 逐笔委托 | 🟢 低 |

> **前置条件**：Tushare 付费层分钟数据权限 + 全市场分钟数据缓存。

### Phase 7: 回测执行器

> **定位**：从因子评估升级到策略回测，桥接研究与实盘。

| 模块 | 内容 | 优先级 |
|------|------|--------|
| 事件驱动引擎 | 日频/分钟频信号 → 订单生成 → 撮合 → 绩效 | 🔴 高 |
| 交易成本模型 | 固定成本 / 滑点 / 市场冲击 / A 股印花税 | 🔴 高 |
| A 股约束 | T+1 / 涨跌停 / 停牌 / 最小交易单位 | 🔴 高 |
| 绩效归因 | Brinson / 因子收益归因 | 🟡 中 |

### Phase 8: 生产就绪

> **定位**：将研究框架推向可用产品状态。

| 模块 | 内容 | 优先级 |
|------|------|--------|
| 性能优化 | 大因子库批量计算 / 并行化 | 🟡 中 |
| 因子数据管理 | 因子存储 / 版本控制 / 血缘追踪 | 🟡 中 |
| CI/CD | GitHub Actions / 自动测试 / lint | 🟡 中 |
| 文档 | API docs / 使用指南 / Jupyter notebooks | 🟢 低 |

### Phase 9: 机器学习增强（可选）

> **定位**：引入 ML 方法进行因子挖掘和非线性合成。

| 模块 | 内容 | 优先级 |
|------|------|--------|
| ML 因子挖掘 | XGBoost / LightGBM 特征重要性筛选 | 🟢 低 |
