# FactorZen Brand Assets

## Logo Concept

The FactorZen mark is a **Factor Grid**: a minimal two-by-two research surface with one green selected point. It represents factor work as a small, repeatable screening process where one candidate survives validation.

This design replaces the denser factor lattice. It removes the validation beam, vermilion signal cell, tagline, bottom color bars, `FZ` monogram, Zen circle, candlestick imagery, trading arrows, coin symbols, generic chart bars, verification stamps, and decorative finance cliches.

## Files

- `logo-horizontal-light.svg`: README and light backgrounds.
- `logo-horizontal-dark.svg`: dark report covers, slides, and dark UI surfaces.
- `logo-mark-light.svg`: compact mark for light backgrounds.
- `logo-mark-dark.svg`: compact mark for dark backgrounds.
- `logo.svg`: compatibility alias for the light horizontal logo.
- `logo-mark.svg`: compatibility alias for the light mark.
- `favicon.svg`: dark square favicon.

## Brand Colors

| Role | Hex | Usage |
| --- | --- | --- |
| Graphite Ink | `#11161A` | Primary mark, wordmark, and high-contrast text |
| Field White | `#F8FAF6` | Dark-mode mark and clean report surfaces |
| Proof Green | `#3E8F66` | The single selected factor point on light backgrounds |
| Signal Green | `#69C88F` | The selected factor point on dark backgrounds and favicon |
| Audit Gray | `#69736E` | Secondary text and small metadata |

## Font Recommendations

- Latin UI / wordmark: `Aptos Display`, `Aptos`, `Segoe UI`.
- Chinese UI: `Noto Sans SC`, `Microsoft YaHei`, `Source Han Sans SC`.
- Code, run IDs, and manifest metadata: `JetBrains Mono`, `Consolas`, monospace.

Use a heavy wordmark for the project name and regular or medium weights for interface copy. Keep letter spacing at `0` for normal text.

## Usage Notes

- Use the light logo on white or neutral light backgrounds.
- Use the dark logo on graphite, dark report covers, or slide backgrounds.
- Use `favicon.svg` for browser tabs and small app icons.
- Keep the green point singular; repeating it weakens the selected-signal metaphor.
- Do not pair the logo with candlestick charts, trading arrows, coin icons, bull/bear symbols, glossy gradients, brush circles, ornamental Zen motifs, generic chart bars, or stamp-style badges.
