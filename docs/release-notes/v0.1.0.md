# v0.1.0 Research Baseline

发布日期：2026-05-19

## 定位

本版本是单因子研究框架的稳定基线，覆盖因子计算、预处理、IC/回测评估、报告生成和实验性多因子合成。它不是生产交易系统，也不承诺组合权重、交易成本和实盘执行的生产级闭环。

## 关键修复

- 修正日频前向收益口径：优先使用 `close[t+h] / close[t] - 1` 的持有期收益，避免把 `ret[t]` 错当未来一期收益。
- 修正日内前向收益：分钟级 `fwd_ret_*bar` 按 `ts_code + trade_date` 分组计算，避免跨交易日取下一根 bar。
- 修正日内预处理：分钟线补齐只在单个交易日内前向填充，不跨日延续旧价格。
- 修复组合 CLI：因子类正确实例化，并在合成评估前准备前向收益输入。
- 恢复质量门：lint、typecheck、pytest 和 smoke 注册表检查恢复为发布前必跑项。

## 已知边界

- `daily/combination/` 当前仍是实验性研究工具，`ic_weighted` 和 `max_ir` 使用样本内 IC 估计权重，不能解释为无偏样本外组合表现。
- `tick/` 仍为预留模块。
- 日内模块已具备评估管线，但真实分钟数据覆盖度仍取决于本地数据源。
