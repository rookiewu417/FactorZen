# FactorZen Logo Redesign

Date: 2026-06-05
Status: implemented

## Context

FactorZen is a reproducible research framework for low-frequency A-share factor evaluation. The project emphasizes trustworthy data handling, IC and layered backtest validation, walk-forward checks, experiment manifests, and HTML tear sheet reports.

## Direction

Use the **Factor Grid + Single Accent** direction. The mark is a two-by-two factor screening surface with one green selected point. The visual tone is intentionally quiet: graphite, field white, and one proof-green accent.

## Visual System

- **Mark:** a rounded square frame containing four circular factor points.
- **Selection:** one green point marks the factor that survives validation.
- **Tone:** calm, rigorous, technical, and evidence-first.
- **Color:** graphite ink, field white, proof green, signal green, and audit gray.
- **Wordmark:** `FactorZen` only, with no tagline or bottom color bars in the SVG lockup.
- **Avoid:** dense lattices, vermilion cells, validation beams, `FZ` monograms, Zen circles, K-line imagery, trading arrows, coin symbols, audit brackets, bar charts, verification stamps, and generic fintech imagery.

## Required Assets

- `docs/assets/logo-mark-light.svg`: mark for light backgrounds.
- `docs/assets/logo-mark-dark.svg`: mark for dark backgrounds.
- `docs/assets/logo-horizontal-light.svg`: README and light-background horizontal lockup.
- `docs/assets/logo-horizontal-dark.svg`: dark-background horizontal lockup.

- `docs/assets/logo-mark.svg`: compatibility alias for the light mark.

## Success Criteria

- The icon remains recognizable at favicon size.
- The README keeps its existing logo reference and displays a cleaner horizontal logo.
- All assets remain text-based SVGs with no external image dependency.
- The design supports the project message: reproducible factor research, validated signals, and trustworthy evidence.
- The logo avoids dense finance cliches and keeps the visual system minimal.
